# Interest Calculator

A simple program to calculate **Simple Interest** and **Total Amount** based on user inputs.  
This project is great for beginners learning basic arithmetic operations and user input handling.

---

##  Features
- Calculate **Simple Interest (SI)**
- Calculate **Total Amount** = Principal + SI
- User-friendly input prompts
- Easy to modify and extend
